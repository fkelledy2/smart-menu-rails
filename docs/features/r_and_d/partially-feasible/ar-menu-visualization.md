# AR Menu Visualization

**Status:** Research / Strategy
**Feasibility:** Partially Feasible / Constrained by Current Platforms
**Target:** 2026+

## Vision

Guests can preview dish size, presentation, and bottles in augmented reality directly on the table.

## Included Ideas

- Camera mode
- 3D dish previews
- Wine bottle size visualization
- Table-scale AR placement

## Feasible Now

- Lightweight 3D previews for selected dishes
- AR-capable mobile experiences where platform/browser support exists
- “View in your space” flows for flagship dishes or bottles

## Constraints

- WebXR support is inconsistent across platforms
- High-quality 3D asset production is expensive
- UX value depends heavily on content quality and device support

## Suggested R&D Path

- Start with premium or high-margin items
- Treat AR as an upsell and confidence tool, not the main browsing model
- Expand only if guests find it genuinely useful

## Strategic Value

- Strong novelty and shareability
- Helps communicate portion size and presentation
- Best suited to visually rich dishes and drinks
