# Smart Table Entry Points

**Status:** Research / Strategy
**Feasibility:** Currently Feasible
**Target:** 2026+

## Vision

Replace QR-heavy initiation flows with faster, lower-friction physical touchpoints.

## Included Ideas

- NFC smart table surfaces
- Tap-to-open table context

## Feasible Now

- NFC tags embedded in tables or holders
- Deep-linked menu opening with signed table context
- Faster handoff with less visual clutter than QR

## Strategic Value

- High UX win with relatively low R&D risk
- Good stepping stone toward more advanced proximity systems
- Reduces reliance on visible QR artifacts

## Suggested R&D Path

- Treat NFC as the immediate successor to QR
- Pair with signed table identity to reduce spoofing
- Roll out by venue type to validate operational durability
