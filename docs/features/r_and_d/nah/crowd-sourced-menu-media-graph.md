# Crowd-Sourced Menu Media Graph

**Status:** Research / Strategy
**Feasibility:** Currently Feasible
**Target:** 2026+

## Vision

Use diner-submitted media to build a living, trustworthy visual layer for dishes and drinks.

## Included Ideas

- Crowd-sourced dish photography
- Real photos from diners tonight

## Feasible Now

- Photo upload flows
- Moderation pipeline
- Ranking by recency, quality, and trust
- Menu card enrichment with provenance such as “real diner photo”

## Constraints

- Requires moderation and abuse prevention
- Restaurant approval workflows may matter for brand-sensitive venues

## Strategic Value

- Powerful data network effect
- More authentic than stock imagery
- Improves guest confidence and discovery

## Suggested R&D Path

- Start with opt-in uploads and moderation queues
- Prioritize high-confidence media and trusted contributors
- Expand toward richer social and recency signals once quality is stable
