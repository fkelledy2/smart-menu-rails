# Nearby Menu Propagation

**Status:** Research / Strategy
**Feasibility:** Requires External Tech Advancement
**Target:** Future / TBD

## Vision

If one guest opens the menu, nearby phones can receive a prompt to join the same table or dining session automatically.

## Included Ideas

- Viral menu propagation
- Nearby join prompts
- AirDrop-style restaurant menu sharing
- Passive table-session discovery for nearby guests

## Why This Is Not Broadly Buildable Yet

- Nearby device discovery is not broadly available to browser apps
- Cross-device trust and consent models are difficult in shared physical spaces
- OS-level sharing systems are not open enough for a portable web implementation

## What Can Be Done Beforehand

- Build share-link based invite flows today
- Use QR, NFC, and server-mediated participant invites
- Prepare the collaboration model so discovery can be layered in later

## Strategic Value

- Could make shared table onboarding feel effortless
- Reduces setup friction for group ordering
- Strengthens the collaborative dining model if ecosystem support improves
