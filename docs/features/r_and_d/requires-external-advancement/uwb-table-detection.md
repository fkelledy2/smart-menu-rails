# UWB Table Detection

**Status:** Research / Strategy
**Feasibility:** Requires External Tech Advancement
**Target:** Future / TBD

## Vision

Use Ultra Wideband-capable phones and table-side hardware so a guest device can determine table proximity with near-physical precision.

## Included Ideas

- Ultra Wideband table detection
- Precise table confirmation at centimeter-level distance
- “Bring food here” confirmation
- More reliable automatic bill splitting and table-aware service routing

## Why This Is Not Broadly Buildable Yet

- UWB is not broadly exposed to browser apps
- Device support is still uneven across markets and platforms
- Restaurants would need dedicated table-side hardware rollout
- Secure, standardized proximity APIs are not yet mainstream on the web

## What Can Be Done Beforehand

- Use NFC and QR as trust anchors today
- Prototype native-app experiments where hardware access is possible
- Define product flows that can later benefit from precise proximity confirmation

## Strategic Value

- Potentially magical UX
- Stronger anti-spoofing and table identity confidence
- Foundational for higher-trust restaurant-floor coordination
